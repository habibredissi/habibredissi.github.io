# Retro Personal Website Design

## Overview
Single-page personal resume/CV website for Habib Redissi. Full retro GeoCities/MySpace/Hi5 aesthetic. Plain HTML + CSS + JS, no frameworks.

## Tech
- `index.html` + `style.css` + `script.js`
- Google Fonts: "Press Start 2P" (headings), "VT323" (body)
- No build tools, no dependencies

## Color Palette
- Background: Light grey (#C0C0C0) with tiled pattern
- Container: White with chunky 2px solid borders, inset box shadows
- Header bar: Hi5 blue gradient (#1E3A8A to #3B82F6)
- Links: Classic blue (#0000EE), visited purple (#551A8B)
- Accents: Yellow (#FFD700), red (#FF0000)
- Buttons: Grey beveled 3D (Windows 98 style)

## Sections
1. **Header/Hero** - Blue gradient bar, pixel font "Welcome to Habib's Homepage", marquee, nav links
2. **About Me** - Forum profile card, photo frame, bio, blinking status dot
3. **Skills** - RPG stat bars (colored HP bars per category)
4. **Experience** - Bulletin board forum posts (Nota, Cast, Contractor, older roles toggle)
5. **Projects** - MySpace Top Friends grid, Windows 98 window cards (Citron, Draft, Proof, Chrome Extension)
6. **Contact** - Guestbook-styled contact links with icons
7. **Footer** - Visitor counter, retro badges (Netscape, Made with Notepad), under construction animation
